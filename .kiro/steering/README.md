# Steering Files

Arquivos nesta pasta são automaticamente incluídos como contexto quando você conversa com o Kiro.

Use para documentar:
- Padrões de código do time
- Convenções de nomenclatura
- Regras de negócio importantes
- Como rodar testes, builds, deploys
- Qualquer informação que o Kiro deve sempre saber sobre o projeto

Os arquivos podem referenciar outros arquivos usando: `#[[file:caminho/arquivo.ts]]`
